# Dybna
**Дубна/Dybna** - приложение, служащее для изучения ***физики***.  

  + Базовые функции:  
    + Расчёт значений по формуле
    + Запоминание формул с помощью теста  

## Использование  
1. На главном экране выбирается наука (пока что только **Физика**).  
2. Выбирается школьный класс для выбора формулы.  
3. В выпадающем списке выбирается формула, после чего нажимается кнопка `Выбрать`.  
4. Значения подставляются в поля ввода. Если значение неизвестно, оставьте поле пустым.  
5. Нажимается кнопка `Расчёт`.  
   
# Техническая часть
Приложение написано на Python-е, в качестве GUI использовается CustomTkinter.

+ Подключаемые библиотеки
  + CustomTkinter
  + requests
